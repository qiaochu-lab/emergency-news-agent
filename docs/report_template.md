# 泛应急行业情报深度研报模板

> 目标：摒弃宏观景气度和趋势炒作，专注于对高价值信息源的深度挖掘与具体分析，形成“务实、落地、有溯源”的情报分析周报。
> 
> 使用原则：
> - 拒绝空泛的趋势判断和主观预测，全部基于具体的信息源、数据或独立事件。
> - 用“事实 + 深度解析”替代“趋势 + 情绪炒作”。
> - 每个重要信息必须标明来源，并对其可信度、关联性、应急场景落地价值进行分析。

# 4 风险与异常预警

> 抛弃“宏观经济不及预期”、“政策推进不及预期”等套话，只写本周收集到的事实中，暴露出的具体技术落地风险或推进延期信号。

- **技术落地风险**：本周哪些被关注的技术在实际信源中被证明存在缺陷或进展缓慢。
- **政策/采购延期信号**：是否有事实表明某方向的需求被推迟或不及此前宣贯。
- **数据矛盾现象**：多个信源之间是否存在指向矛盾的地方。

---

# 附：写作风格要求

- 语气应客观、冷静、基于证据。
- **事实与判断必须分离**：“提取”部分只写事实，“分析”部分才写推断。
- 切忌盲目乐观或使用营销词汇，保持分析师的批判性视角。
