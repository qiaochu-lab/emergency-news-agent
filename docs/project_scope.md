# Project Scope

## MVP Goal

Deliver one stable weekly Markdown report that transforms multi-source inputs into a decision-oriented Chinese intelligence brief.

## Included in MVP

- Manual run pipeline
- Weekly scheduled run support
- Source adapters for RSS, web, and API style inputs
- Rules-first deduplication, classification, and scoring
- Provider abstraction for structured LLM analysis
- Markdown report generation

## Out of Scope

- Dashboard
- Knowledge graph
- Realtime streaming
- Complex agent orchestration
- Automated source discovery
